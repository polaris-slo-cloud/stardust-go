# Contributing to Stardust

Thank you for considering contributing to Stardust! 

If you want to contribute code, please open a pull request on this GitHub repository. Please make sure that your code compiles with no warnings or errors before submitting your pull request.

If you encounter any bugs, issues, need assistance, or have suggestions for improvements, please open an issue in the [GitHub repository](https://github.com/polaris-slo-cloud/stardust-go/issues). Provide as much detail as possible, including steps to reproduce the issue, expected behavior, and actual behavior.

If your contribution involves changes to the functionality, update the relevant documentation in the `README.md` or other appropriate files.
